# 文献库使用指南 / Literature Library Guide

## 概述 / Overview

本目录收录与 **动态适应与协同演化系统理论（DACES）** 相关的支撑文献、理论演化文档、架构设计文档、协议规范、模型能力分析与路线图文档。

这些文献为 AI Agent 系统架构设计提供理论基础与实践参考，适用于：

- **AI 架构师**：设计可扩展、安全、可治理的 Agent 系统。
- **研究人员**：理解多 Agent 协作、动态适应、协同演化与执行资产治理。
- **开发者**：参考 OpenAdam / OpenCode 类系统的工程实践与架构方法。
- **Agent 系统构建者**：为 Agent 提供可解析、可复用、可审查的理论材料。

## 与 DACES 理论的关系 / Relationship to DACES

本文献库支撑的核心理论框架包括：

| 理论维度 / Dimension | 全称 / Full Name | 支撑文献 / Supporting Documents |
| --- | --- | --- |
| **D** | Dynamic Adaptation（动态适应） | `02_theory_evolution/` |
| **A** | Adaptive Evolution（协同演化） | `01_foundations/1.3_Harness-Multi-Agent-System.md`, `01_foundations/1.6_Multi-Agent-Collaboration-Ecosystem.md`, `01_foundations/1.9_Multi-Agent-Context-Isolation.md` |
| **C** | Collaborative System（协作系统） | `01_foundations/`, `02_theory_evolution/` |
| **E** | Engineering Harness（工程 Harness） | `01_foundations/1.2_Prompt-to-Harness-Engineering.md`, `01_foundations/1.3_Harness-Multi-Agent-System.md` |
| **S** | Safety Architecture（安全架构） | `03_architecture_design/`, `04_protocols_standards/` |

EAPF（Execution Asset Promotion Framework）补充了 DACES 的执行资产治理视角：它关注任务应由哪一级执行资产承接，以及系统如何通过 review、telemetry、promotion 和 demotion 机制避免资产层级堕落。

## 使用说明 / Usage Notes

- 文献按**依赖关系**与**主题类别**组织，建议按推荐阅读路径顺序阅读。
- 所有文档基于 2025-2026 年研究与实践积累。
- 文档格式为 Markdown，支持直接阅读，也适合作为 Agent 或研究助手的上下文材料。
- 对于 Agent 读取，建议优先使用包含摘要、术语表、结构化表格和阅读指南的文档。
