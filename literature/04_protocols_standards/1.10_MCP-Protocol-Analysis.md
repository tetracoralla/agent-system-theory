### 2.2 各组件职责详解

| 组件 | 职责 | 典型实现 | 关键特性 |
|------|------|----------|----------|
| **Host** | AI智能承载 | Claude Desktop、VS Code、Cursor、IntelliJ IDEA | 模型无关、智能决策、上下文管理 |
| **Client** | 协议通信 | MCP SDK（Python/Node.js/Go） | 连接管理、协议解析、错误处理 |
| **Server** | 能力执行 | 文件系统Server、数据库Server、API Server | 资源访问、工具暴露、安全控制 |

### 4.3 Agent-MCP集成示例

```python
class AgentWithMCP:
    """集成MCP的Agent系统"""
    
    def __init__(self, model, mcp_config):
        self.model = model  # AI模型
        self.mcp_client = MCPClient(mcp_config)
        self.available_tools = []
        
    async def initialize(self):
        """初始化：发现可用工具"""
        # 连接MCP服务器
        await self.mcp_client.connect()
        
        # 获取可用工具列表
        self.available_tools = await self.mcp_client.list_tools()
        
        # 注册到Agent
        self.model.register_tools(self.available_tools)
        
    async def execute_task(self, task):
        """执行任务"""
        # 1. Agent规划任务
        plan = await self.model.plan(task)
        
        # 2. 通过MCP调用工具
        for step in plan.steps:
            if step.requires_tool:
                # MCP标准化调用
                result = await self.mcp_client.call_tool(
                    name=step.tool_name,
                    arguments=step.arguments
                )
                step.result = result
        
        # 3. 整合结果
        return await self.model.synthesize(plan)
```
