# 动态适应与协同演化系统理论 · openAdam

---

## 简介 / Introduction

本仓库汇集了关于“动态适应与协同演化系统理论”（DACES）的研究成果。内容以一篇核心方法论研究文档为基础，并延伸出多篇面向企业级工作流、多 Agent 系统工程、执行资产治理、协议规范与社区实践的补充研究。

This repository brings together research on the **Dynamic Adaptation and Collaborative Evolution System Theory (DACES)**. The content is based on a core methodological framework and extends into multiple engineering application theories, execution-asset governance models, protocol specifications, and supplementary studies for enterprise-level multi-agent systems.

这些文档并非由单一作者独立完成，而是在既定研究方向与架构愿景框架下，通过协作流程，包括相互审查与补充，逐步形成的。目标是提供一个可供学术界与工业界参考、讨论与发展的知识基础。

These documents were not independently authored by a single individual but were formed through a collaborative process, including mutual review and supplementation, within a defined research direction and architectural vision framework. The goal is to provide a knowledge base for reference, discussion, and development by academia and industry.

## 快速开始 / Quick Start

### 中文阅读路径

1. **理解理论**：先阅读 `methodological_framework_and_working_paper.md`，了解 DACES 理论的基本视角、核心概念与提出的问题。
2. **探索应用**：接着阅读 `Multi-Agent-System-Engineering-Application-Framework.md`，了解如何将理论应用于企业级多 Agent 系统工程。
3. **进入治理视角**：阅读 `literature/02_theory_evolution/3.9.14_Execution-Asset-Promotion-Framework.zh-CN.md`，快速理解执行资产升级框架 EAPF。
4. **深入研究**：根据兴趣选择阅读设计系统、隐性知识、工作流标准化、协议规范与模型能力分析等补充文献。

### English Reading Path

1. **Understand the Theory**: Start with `methodological_framework_and_working_paper.md` to understand the basic perspective, core concepts, and questions posed by DACES.
2. **Explore Application**: Read `Multi-Agent-System-Engineering-Application-Framework.md` to understand how the theory can be applied to enterprise-level multi-agent system engineering.
3. **Enter the Governance Layer**: Read `literature/02_theory_evolution/3.9.14_Execution-Asset-Promotion-Framework.md` to understand EAPF, a framework for execution asset governance.
4. **Dive Deeper**: Based on specific interests, read the relevant supplementary research papers and literature library documents.
